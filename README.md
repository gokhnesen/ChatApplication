Proje Hakkında

Bu çalışma, modern web teknolojilerini pekiştirmek amacıyla geliştirdiğim kişisel bir projedir. Özellikle Clean Architecture yapısını kurmayı ve katmanlı mimaride gerçek zamanlı iletişim (SignalR) senaryolarını çözmeyi deneyimlediğim bir uygulama oldu.

Temel Özellikler 


Gerçek Zamanlı İletişim: SignalR kullanılarak anlık mesajlaşma (WebSocket).
Güvenli Kimlik Doğrulama: Microsoft Identity altyapısı ile güvenli oturum yönetimi (Cookie-based Authentication).
Clean Architecture: Proje, bağımlılıkları minimize eden ve test edilebilirliği artıran katmanlı mimari (Domain, Application, Infrastructure, Persistence) ile geliştirildi.
Modern Arayüz: Angular ve SCSS ile modern tasarım.
CQRS Pattern: Komut ve sorgu sorumluluklarının ayrıştırılması (MediatR kütüphanesi ile).
